    发布blog流程

    hugo新建post并推送到GitHub page的流程（怕忘了，history也看不到） hugo项目的根目录执行

    hugo new post/Debian重启ssh.md

    会在content/post目录下新建对应md文件，编辑完后，执行

    hugo --theme=hugo-book --baseURL="https://msinwolc.github.io/saysth"

    生成对应静态资源，然后进入public目录

    cd public
git add .
git commit -m 'first commit'
git pull 
git push

    如果要在左边加入导航栏，有一点点麻烦，首先得开启这个功能，然而网上的教程都是旧的，最新版的hugo自动生成的配置文件不叫config.toml，而是hugo.toml，里面只有很简单的三行，需要加上以下部分

    [params]
  BookMenuBundle = '/menu'

    然后在content目录新建menu目录，新建index.md，内容如下

    +++
headless = true
+++

- [Linux](/saysth/post/)
  - [Debian重启ssh](/saysth/post/debian%E9%87%8D%E5%90%AFssh/)

    再执行生成静态资源往后的步骤，部署成功后即可看到导航栏
